# Invoice Processing System

Welcome to the Invoice Processing System! This is a full-stack application designed to manage and streamline the process of invoicing. It utilizes a modern tech stack comprising React, Material UI, Spring Boot, Hibernate, and MySQL. Whether you're a budding developer looking to enhance your skills or a business seeking an efficient invoicing solution, this step-by-step tutorial has got you covered.

## Features

- User-friendly interface with React and Material UI
- Robust backend with Spring Boot and Hibernate
- Persistent data storage with MySQL
- Manage invoices with CRUD operations
- Responsive design for various devices

## Installation

### Prerequisites

- Java 11 or higher
- Node.js and npm
- MySQL

### Backend Setup

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/invoice-processing-system.git
    ```

2. Navigate to the backend directory:
    ```bash
    cd invoice-processing-system/backend
    ```

3. Configure your MySQL database. Create a `application.properties` file in `src/main/resources` and set up your database credentials.

4. Build and run the Spring Boot application:
    ```bash
    ./mvnw spring-boot:run
    ```

### Frontend Setup

1. Navigate to the frontend directory:
    ```bash
    cd ../frontend
    ```

2. Install the required dependencies:
    ```bash
    npm install
    ```

3. Start the React application:
    ```bash
    npm start
    ```

## Usage

Once both the frontend and backend are running, open your browser and go to `http://localhost:3000` to access the Invoice Processing System. You can start creating, viewing, updating, and deleting invoices from the user-friendly interface.
